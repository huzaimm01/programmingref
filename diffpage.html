Coding Commands Reference

C

Basic Commands

  • Compile a C Program: 
    gcc myProgram.c -o myProgram
    Compiles the C program into an executable named `myProgram`.
  • Run a C Program: 
    ./myProgram
    Executes the compiled C program.
  • Include a Header File: 
    #include <stdio.h>
    Includes the standard I/O library for input and output functions.
  • Declare a Function: 
    int add(int a, int b) { return a + b; }
    Defines a function to add two integers.

C++

Basic Commands

  • Compile a C++ Program: 
    g++ myProgram.cpp -o myProgram
    Compiles the C++ program into an executable named `myProgram`.
  • Run a C++ Program: 
    ./myProgram
    Executes the compiled C++ program.
  • Include a Library: 
    #include <iostream>
    Includes the I/O stream library for input and output operations.
  • Defining a Class: 
    class MyClass { public: void myFunction(); };
    Defines a simple class with a public member function.

C#

Basic Commands

  • Create a New C# Project: 
    dotnet new console -n MyApp
    Creates a new console application named `MyApp`.
  • Run the Application: 
    dotnet run
    Runs the currently active C# application.
  • Using a Namespace: 
    using System;
    Includes the `System` namespace for basic functionality.
  • Defining a Method: 
    public void Greet() { Console.WriteLine("Hello!"); }
    Defines a method to print a greeting message.

Java

Basic Commands

  • Compile a Java Program: 
    javac MyProgram.java
    Compiles the Java program into bytecode.
  • Run a Java Program: 
    java MyProgram
    Executes the compiled Java program.
  • Import a Package: 
    import java.util.ArrayList;
    Imports the `ArrayList` class from the `java.util` package.
  • Defining a Class: 
    public class MyClass { public void greet() { System.out.println("Hello!"); } }
    Defines a class with a method to print a greeting message.

JavaScript

Basic Commands

  • Print to Console: 
    console.log("Hello, World!");
    Outputs the string to the console.
  • Declare a Variable: 
    let x = 10;
    Declares a variable `x` and initializes it to 10.
  • Define a Function: 
    function greet() { return "Hello!"; }
    Defines a function that returns a greeting string.
  • Add an Event Listener: 
    button.addEventListener("click", function() { alert("Clicked!"); });
    Attaches a click event listener to a button.

HTML

Basic Tags

  • HTML Document Structure: 
    <!DOCTYPE html><html></html>
    Defines the basic structure of an HTML document.
  • Creating a Link: 
    <a href="https://example.com">Link</a>
    Creates a hyperlink to `example.com`.
  • Creating an Image: 
    <img src="image.jpg" alt="Description">
    Embeds an image in the document.
  • Creating a List: 
    <ul><li>Item 1</li></ul>
    Creates an unordered list with one item.

CSS

Basic Properties

  • Change Text Color: 
    color: red;
    Sets the text color to red.
  • Change Background Color: 
    background-color: blue;
    Sets the background color to blue.
  • Set Font Size: 
    font-size: 16px;
    Sets the font size to 16 pixels.
  • Set Margin: 
    margin: 10px;
    Sets a margin of 10 pixels.

Shell (Bash)

Basic Commands

  • List Files: 
    ls -l
    Lists files in the current directory with details.
  • Change Directory: 
    cd /path/to/directory
    Changes the current directory to the specified path.
  • Copy a File: 
    cp source.txt destination.txt
    Copies `source.txt` to `destination.txt`.
  • Remove a File: 
    rm file.txt
    Deletes `file.txt`.

SQL

Basic Commands

  • Select Data: 
    SELECT * FROM table_name;
    Retrieves all records from the specified table.
  • Insert Data: 
    INSERT INTO table_name (column1, column2) VALUES (value1, value2);
    Inserts a new record into the specified table.
  • Update Data: 
    UPDATE table_name SET column1 = value1 WHERE condition;
    Updates existing records in the specified table.
  • Delete Data: 
    DELETE FROM table_name WHERE condition;
    Deletes records from the specified table based on the condition.
