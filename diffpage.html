Coding Commands Reference

C

Basic Commands

  • Compile a C Program: 
    gcc myProgram.c -o myProgram
    Compiles the C program into an executable named `myProgram`.
  • Run a C Program: 
    ./myProgram
    Executes the compiled C program.
  • Include a Header File: 
    #include <stdio.h>
    Includes the standard I/O library for input and output functions.
  • Declare a Function: 
    int add(int a, int b) { return a + b; }
    Defines a function to add two integers.

C++

Basic Commands

  • Compile a C++ Program: 
    g++ myProgram.cpp -o myProgram
    Compiles the C++ program into an executable named `myProgram`.
  • Run a C++ Program: 
    ./myProgram
    Executes the compiled C++ program.
  • Include a Library: 
    #include <iostream>
    Includes the I/O stream library for input and output operations.
  • Defining a Class: 
    class MyClass { public: void myFunction(); };
    Defines a simple class with a public member function.

C#

Basic Commands

  • Create a New C# Project: 
    dotnet new console -n MyApp
    Creates a new console application named `MyApp`.
  • Run the Application: 
    dotnet run
    Runs the currently active C# application.
  • Using a Namespace: 
    using System;
    Includes the `System` namespace for basic functionality.
  • Defining a Method: 
    public void Greet() { Console.WriteLine("Hello!"); }
    Defines a method to print a greeting message.

Java

Basic Commands

  • Compile a Java Program: 
    javac MyProgram.java
    Compiles the Java program into bytecode.
  • Run a Java Program: 
    java MyProgram
    Executes the compiled Java program.
  • Import a Package: 
    import java.util.ArrayList;
    Imports the `ArrayList` class from the `java.util` package.
  • Defining a Class: 
    public class MyClass { public void greet() { System.out.println("Hello!"); } }
    Defines a class with a method to print a greeting message.

Arduino

Basic Commands

  • Initialize Serial Communication: 
    Serial.begin(9600);
    Starts serial communication at 9600 bps.
  • Declare a Variable: 
    int x = 10;
    Declares an integer variable x.
  • If Statement: 
    if (x > 10) { Serial.println("Greater"); }
    Executes if x is greater than 10.
  • For Loop: 
    for (int i = 0; i < 5; i++) { Serial.println(i); }
    Iterates from 0 to 4 and prints the value.

Raspberry Pi

Basic Commands

  • Print to Console: 
    print("Hello, World!")
    Prints "Hello, World!" to the console.
  • Declare a Variable: 
    x = 10
    Declares a variable x.
  • If Statement: 
    if x > 10:\n    print("Greater")
    Executes if x is greater than 10.
  • For Loop: 
    for i in range(5):\n    print(i)
    Iterates from 0 to 4 and prints the value.

MongoDB

Basic Commands

  • Find Documents: 
    db.collection.find({name: 'Alice'})
    Finds documents with the name Alice.
  • Insert Document: 
    db.collection.insert({name: 'Alice', age: 30})
    Inserts a new document for Alice.
  • Update Document: 
    db.collection.update({name: 'Alice'}, {$set: {age: 31}})
    Updates Alice's age to 31.
  • Delete Document: 
    db.collection.remove({name: 'Alice'})
    Removes documents with the name Alice.

MySQL

Basic Commands

  • Select Data: 
    SELECT * FROM users;
    Retrieves all data from the users table.
  • Insert Data: 
    INSERT INTO users (name, age) VALUES ('Alice', 30);
    Inserts a new user named Alice.
  • Update Data: 
    UPDATE users SET age = 31 WHERE name = 'Alice';
    Updates Alice's age to 31.
  • Delete Data: 
    DELETE FROM users WHERE name = 'Alice';
    Deletes the user named Alice.
