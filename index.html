Python Commands

Python is a versatile programming language known for its simplicity and readability.

Basic Syntax:
print: Outputs to the console.
print("Hello, World!")  # Displays "Hello, World!" on the screen.

Variables:
x = 10: Assigns value 10 to variable x.
name = "Alice": Assigns string "Alice" to variable name.

Control Structures:
if, else: Conditional statements that execute based on a condition.
if x > 10:
    print("Greater")  # Executes if x is greater than 10.
else:
    print("Smaller")  # Executes if x is not greater than 10.

for: Looping structure to iterate over a sequence.
for i in range(5):  # Iterates from 0 to 4.
    print(i)

Functions:
def: Defines a function to encapsulate code for reusability.
def greet(name):
    print("Hello, " + name)  # Greets the user with their name.

Lists:
my_list = [1, 2, 3]: Creates a list with three integers.
my_list.append(4): Adds 4 to the end of my_list.

Go Commands

Go, also known as Golang, is a statically typed language developed by Google.

Basic Syntax:
fmt.Println: Outputs to the console.
fmt.Println("Hello, World!")  # Prints "Hello, World!" to the console.

Variables:
var x int = 10: Declares a variable x of type int and initializes it to 10.
name := "Alice": Short variable declaration for string type.

Control Structures:
if, else: Conditional statements that execute based on a condition.
if x > 10 {
    fmt.Println("Greater")  # Executes if x is greater than 10.
} else {
    fmt.Println("Smaller")  # Executes if x is not greater than 10.
}

for: Looping structure to iterate.
for i := 0; i < 5; i++ {  # Iterates from 0 to 4.
    fmt.Println(i)
}

Functions:
func: Defines a function for reusable code.
func greet(name string) {
    fmt.Println("Hello, " + name)  # Greets the user with their name.
}

SQL Commands

SQL (Structured Query Language) is used for managing and querying relational databases.

Basic Syntax:
SELECT: Retrieves data from a database.
SELECT * FROM users;  # Selects all columns from the users table.

INSERT: Adds new data to a database.
INSERT INTO users (name, age) VALUES ('Alice', 30);  # Adds a new user.

UPDATE: Modifies existing data in a database.
UPDATE users SET age = 31 WHERE name = 'Alice';  # Updates Alice's age.

DELETE: Removes data from a database.
DELETE FROM users WHERE name = 'Alice';  # Deletes Alice from the users table.

HTML Commands

HTML (Hypertext Markup Language) is the standard markup language for creating web pages.

Basic Syntax:
<html>: Root element of an HTML page.
<head>: Contains meta-information about the document.
<body>: Contains the content of the document.

Elements:
<h1>: Defines a top-level heading.
<p>: Defines a paragraph of text.
<a href="https://example.com">: Creates a hyperlink to another page.

Attributes:
src: Specifies the URL of an image.
<img src="image.jpg" alt="Description">  # Displays an image with an alternative text.

PHP Commands

PHP (Hypertext Preprocessor) is a server-side scripting language for web development.

Basic Syntax:
echo: Outputs to the console.
echo "Hello, World!";  # Prints "Hello, World!" to the console.

Variables:
$var: Declares a variable.
$age = 30;: Assigns value 30 to variable age.

Control Structures:
if, else: Conditional statements that execute based on a condition.
if ($age > 18) {
    echo "Adult";  # Executes if age is greater than 18.
} else {
    echo "Minor";  # Executes if age is not greater than 18.
}

for: Looping structure to iterate.
for ($i = 0; $i < 5; $i++) {
    echo $i;  # Outputs numbers from 0 to 4.
}

Functions:
function: Defines a function for reusable code.
function greet($name) {
    echo "Hello, " . $name;  # Greets the user with their name.
}

CSS Commands

CSS (Cascading Style Sheets) is used for styling web pages.

Basic Syntax:
selector { property: value; }: Defines styles for HTML elements.
h1 { color: red; }  # Changes h1 text color to red.

Classes and IDs:
.class: Applies styles to elements with a specific class.
#id: Applies styles to a specific element with an ID.
.container { margin: 20px; }  # Styles elements with class "container".
#header { background-color: blue; }  # Styles the element with ID "header".

Box Model:
width: Sets the width of an element.
padding: Space between content and border.
margin: Space outside the element.

MongoDB Commands

MongoDB is a NoSQL database that uses documents in BSON format.

Basic Syntax:
db.collection.find: Retrieves documents from a collection.
db.users.find({name: 'Alice'});  // Finds users with the name Alice.

db.collection.insert: Adds a new document to a collection.
db.users.insert({name: 'Alice', age: 30});  // Inserts a new user.

db.collection.update: Modifies an existing document.
db.users.update({name: 'Alice'}, {$set: {age: 31}});  // Updates Alice's age.

db.collection.remove: Deletes a document from a collection.
db.users.remove({name: 'Alice'});  // Deletes the user named Alice.
